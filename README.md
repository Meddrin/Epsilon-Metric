# Epsilon-Metric
This function calculates the epsilon metric (epsilon indicator) of two given sets.

```math
I_\epsilon (A,B) =\max_{b \in B} \min_{a \in A} \max_{1 \leq i \leq n} \frac{a_i}{b_i}
```
